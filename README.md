# MyDoc - Offline markdown-based editor in the browser

An open-source markdown based editor designed for simplicity. Offers a clean, minimal writing space. Everything is autosaved locally and never leaves your device, with options to export as PDF, DOCX, or RTF. 

**Technologies**: Vercel, NextJS, TypeScript

**Live Demo**: https://mydocs.emjjkk.tech
